Namespace: dom0

ax5. dom0

Refer to this by ax5.
dom0는 dom(query)를 줄여서 칭하는 말 입니다. ax5.dom 에 있는 함수들과는 다른 개념입니다. ax5.dom0는 query에 의해 만들어진 인스턴스 입니다.

Example

ax5.dom("#elementid");
var ax = ax5.dom;
ax("#elementid");
// 처러 사용할 수 있습니다.
