CSS

AX5 CSS 사용법을 만나실 수 있습니다.

ax

AX5 CSS 시스템의 기본이 되는 class="ax" 입니다. ax로 시작하는 CSS클래스는 폰트사이즈, 글꼴, 굵기, 줄높이를 결정하여줍니다. ax에 basic, good등의 상황값을 추가 하면 상황에 맞는 색상으로 글자 색상을 변경 하거나. lg, sm, xs 등의 크기값을 추가하여 크기를 변경할 수 있습니다.

basic

<a class="ax" href="#">AX5 ax5 에이액스5</a>
<div class="ax">AX5 ax5 에이액스5</div>
<div class="ax">
	AX5 ax5 에이액스5
	<span class="badge">5</span>
</div>

Result

AX5 ax5 에이액스5
AX5 ax5 에이액스5
AX5 ax5 에이액스5 5
